# Security

## Reporting a Vulnerability

If you discover a security issue in this project, please **do not open a public issue**.

Instead, send a private report by opening a GitHub Security Advisory at:
https://github.com/iknowkungfubar/property-management-leadgen/security/advisories/new

We will acknowledge receipt within 48 hours and provide an expected timeline
for a fix and disclosure.

## Scope

The following are in scope for security reports:

- Remote code execution vectors
- Authentication/authorization bypass
- SQL injection, path traversal, or SSRF in production paths
- Credential leakage through the application
- Supply chain vulnerabilities in pinned dependencies

The following are **out of scope**:

- Dependency vulnerabilities already tracked by Dependabot (see alerts page)
- Missing docstrings or type annotations
- Theoretical attacks requiring physical access to the user's machine
